# Basurero-Inteligente-TEI201
Un basurero inteligente que avisa cuando este lleno para evitar contaminación.
Sistema IOT que mide el porcentaje de llenado de un basurero en espacios compartidos de la universidad y genera una alerta en una pantalla OLED para escanear un QR cuando llega al límite programado con el fin de avisar a personal de limpieza evitando contaminación y un mal ambiente.
